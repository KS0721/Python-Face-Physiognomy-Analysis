# 🐍 Python-Face-Physiognomy-Analysis

> **카메라 기반 관상 분석 데스크톱 애플리케이션 (초년/중년/말년 운세)**

![Python, OpenCV, MediaPipe를 활용한 관상 분석 앱의 실행 예시 이미지] 
 

본 프로젝트는 **파이썬(Python)**과 **컴퓨터 비전(Computer Vision)** 기술을 활용하여 사용자의 얼굴 특징점(Landmark)을 실시간으로 분석하고, 관상학적 규칙에 기반한 운세 풀이를 제공하는 **데스크톱 GUI 애플리케이션**입니다.

이 프로젝트는 **OpenCV, MediaPipe, Tkinter**를 활용하여 데이터 처리부터 GUI 구현까지 통합된 파이프라인을 구축한 개인 포트폴리오 결과물입니다.

---

## 🎯 주요 기능 및 분석 구조

### 1. 주요 기능

* **실시간 스트리밍:** 웹캠을 활용한 실시간 영상 프레임 디스플레이.
* **정밀 특징점 추출:** **MediaPipe Face Mesh**를 사용하여 얼굴의 468개 주요 지점 좌표를 정밀하게 감지.
* **운세 분석:** 추출된 특징점 간의 **비율 및 길이**를 계산하여 **초년(이마/눈), 중년(코), 말년(턱)** 운세 분석 결과를 한국어로 제공.
* **GUI 인터페이스:** Tkinter를 활용하여 사용자가 쉽게 접근하고 결과를 확인할 수 있는 단독 실행 앱 환경 제공.

### 2. 관상학적 분석 구조

관상학에서는 얼굴을 **상정(上停), 중정(中停), 하정(下정)** 세 부분으로 나누어 운세를 판단합니다. 

| 운세 시기 | 관상 영역 | 분석 부위 | 운세 해석 |
| :--- | :--- | :--- | :--- |
| **초년운** | **상정** (이마~눈썹) | 눈 간 거리, 이마의 폭 | 학업, 초년의 성공, 대인관계 |
| **중년운** | **중정** (눈썹~코 끝) | 코의 너비, 코의 길이 | **재물운**, 직업운, 중년의 안정성 |
| **말년운** | **하정** (코 끝~턱 끝) | 턱의 길이 및 모양 | 건강, 말년의 복, 자손과의 관계 |
